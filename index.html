·HTML

HTML stands for Hypertext Markup Language. Tim Berners-Lee wrote the first version of HTML in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000s was HTML 4.01, which became an official standard in December 1999. At present the most used version of HTML is HTML5. Another version, XHTML, was a rewrite of HTML as an XML language.

HTML was invented initially as a language intended for the distribution of relatively simple for use by authors documents, suitable or academics rather than experts in printing or other aspects of document formatting.
